# one-bit-adc-dac



## Channel Attributes

### voltage0

| Name | sysfs filename | Default value |
| ---- | -------------- | ------------- |
| label | out_voltage0_label |  |
| raw | out_voltage0_raw |  |


**iio_attr example:**

```bash
iio_attr -u ip:analog -c one-bit-adc-dac voltage0 label
```

**Python bindings example:**

```python
import iio
ctx = iio.Context("ip:analog")
dev = ctx.find_device("one-bit-adc-dac")
chan = dev.find_channel("voltage0")
rval = chan.attrs["label"].value
```
