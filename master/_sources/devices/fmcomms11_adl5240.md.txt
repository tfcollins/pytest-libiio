# adl5240



## Channel Attributes

### voltage0

| Name | sysfs filename | Default value |
| ---- | -------------- | ------------- |
| hardwaregain | out_voltage0_hardwaregain |  |


**iio_attr example:**

```bash
iio_attr -u ip:analog -c adl5240 voltage0 hardwaregain
```

**Python bindings example:**

```python
import iio
ctx = iio.Context("ip:analog")
dev = ctx.find_device("adl5240")
chan = dev.find_channel("voltage0")
rval = chan.attrs["hardwaregain"].value
```
