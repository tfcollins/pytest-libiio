## Channel Attributes

### voltage0

| Name | sysfs filename | Default value |
| ---- | -------------- | ------------- |
| raw | in_voltage0_raw |  |
| scale | in_voltage_scale |  |


**iio_attr example:**

```bash
iio_attr -u ip:analog -c ltc2308 voltage0 raw
```

**Python bindings example:**

```python
import iio
ctx = iio.Context("ip:analog")
dev = ctx.find_device("ltc2308")
chan = dev.find_channel("voltage0")
rval = chan.attrs["raw"].value
```
