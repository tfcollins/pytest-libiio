# ad7291



## Channel Attributes

### temp0

| Name | sysfs filename | Default value |
| ---- | -------------- | ------------- |
| mean_raw | in_temp0_mean_raw |  |
| raw | in_temp0_raw |  |
| scale | in_temp0_scale |  |


**iio_attr example:**

```bash
iio_attr -u ip:analog -c ad7291 temp0 mean_raw
```

**Python bindings example:**

```python
import iio
ctx = iio.Context("ip:analog")
dev = ctx.find_device("ad7291")
chan = dev.find_channel("temp0")
rval = chan.attrs["mean_raw"].value
```
