# ad9517-3



## Channel Attributes

### altvoltage0

| Name | sysfs filename | Default value |
| ---- | -------------- | ------------- |
| frequency | out_altvoltage0_frequency |  |
| raw | out_altvoltage0_raw |  |


**iio_attr example:**

```bash
iio_attr -u ip:analog -c ad9517-3 altvoltage0 frequency
```

**Python bindings example:**

```python
import iio
ctx = iio.Context("ip:analog")
dev = ctx.find_device("ad9517-3")
chan = dev.find_channel("altvoltage0")
rval = chan.attrs["frequency"].value
```
