**iio_attr example:**

```bash
iio_attr -u ip:analog -c axi-core-tdd data0 dp_off_ms
```

**Python bindings example:**

```python
import iio
ctx = iio.Context("ip:analog")
dev = ctx.find_device("axi-core-tdd")
chan = dev.find_channel("data0")
rval = chan.attrs["dp_off_ms"].value
```
